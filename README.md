# Movie Translator

A React Native app that finds translated movie titles in 29 languages using The Movie Database API.

## Setup

```bash
npm install
npm start
```

## Usage

1. Select a target language from the dropdown
2. Search for a movie by its English title
3. View the translated title in your selected language

## Scripts

- `npm start` - Start the Expo development server
- `npm run ios` - Run on iOS simulator
- `npm run android` - Run on Android emulator
- `npm run web` - Run in web browser
